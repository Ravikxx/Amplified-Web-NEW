About the Server

Amplified SMP is a Java & Bedrock crossplay anarchy survival server running on Paper 1.21.1, hosted on Apect. No scripting, no rules on how you play — grief, steal, PvP, do whatever. The world is yours.

The server has a solid plugin stack including DiscordSRV for chat bridging, CoreProtect for rollbacks, LuckPerms for permissions, Geyser/Floodgate for Bedrock support, and Simple Voice Chat for proximity voice.
